HellicarAndLewis

This application is to be used with the 006 version (http://openframeworks.cc/download) of OpenFrameworks.

The application was created with the Victoria and Albert Museum Education Department, to create an educational event for the Baroque Exhibition.

http://www.vam.ac.uk/microsites/baroque/

Press space bar to take a snap! First open up faces.xml in the data folder and edit the line:

<SAVELOCATION>/Users/joel/Documents/Dropbox/HellicarAndLewisDrop/newFaces/</SAVELOCATION>

to be something that makes sense on your computer.

F to go fullscreen.

OS X version tested only.
